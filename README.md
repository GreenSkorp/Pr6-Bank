В процессе тестирования:
- создан и задокументирован тестируемый проект;
- разработаны модульные тесты для проверки корректности работы методов;
- выявлена и исправлена ошибка в методе Debit (неверный знак операции);
- выполнено тестирование граничных и некорректных значений;
- проведён рефакторинг кода и тестов.

## Итоги тестирования

Все тесты успешно пройдены, что подтверждается результатами обозревателя тестов.
Причина успешного выполнения тестов: после исправления логической ошибки в методе Debit (замена += на -=) и корректной обработки исключений тестовые сценарии полностью соответствуют ожидаемому поведению программы.

Тестирование позволило повысить надёжность программного модуля и убедиться в корректности его работы.
